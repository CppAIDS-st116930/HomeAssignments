## Description
This program implements a Reverse Polish Notation (RPN) calculator using a custom stack implementation.
It supports basic arithmetic operations: `+`, `-`, `*`, `/`.

## Build
```bash
make
```

## Run
```bash
./bin/rpn
```

## Testing
Unit tests are implemented using GoogleTest.
To run tests:
```bash
make test
```
